# 📈 株価チェッカー

シンプルで使いやすい株価情報表示アプリです。

## 機能

### 🔍 銘柄検索
- 銘柄コードを入力して株価情報を検索
- 日本株（例：7203）および米国株（例：AAPL）に対応
- エンターキーでの検索にも対応

### 📊 人気銘柄
ワンクリックで以下の人気銘柄の株価をチェック：
- **トヨタ自動車** (7203)
- **ソフトバンクグループ** (9984)
- **ソニーグループ** (6758)
- **ファーストリテイリング** (9983)
- **Apple** (AAPL)
- **Google** (GOOGL)

### 💹 表示情報
- 現在の株価
- 価格変動（金額・パーセント）
- 上昇・下落の色分け表示

## 技術仕様

### フロントエンド
- **HTML5** + **CSS3** + **JavaScript**
- レスポンシブデザイン
- グラデーション背景とモダンなUI

### API連携
- Finnhub APIを使用（デモキー）
- APIエラー時はデモデータを表示
- 非同期データ取得

### デザイン特徴
- **カラーパレット**: ブルー系グラデーション
- **アニメーション**: ホバー効果、ローディングスピナー
- **レイアウト**: カード形式、グリッドレイアウト

## 使用方法

1. ブラウザで `index.html` を開く
2. 銘柄コードを入力して「検索」ボタンをクリック
   - または人気銘柄から選択
3. 株価情報が表示される

## 注意事項

- デモ用APIキーを使用しているため、実際のリアルタイムデータは制限されます
- 実用化する場合は有効なFinnhub APIキーを取得してください
- デモデータが表示される場合があります
