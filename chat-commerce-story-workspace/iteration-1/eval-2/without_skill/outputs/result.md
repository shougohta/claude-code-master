## 1. サービス概要

**BRUNO online（ブルーノオンライン）** は、ライフスタイルブランド「BRUNO」が運営する公式ECサイトです。キッチン家電・インテリア雑貨を中心に、ファッション・トラベル用品・ビューティー製品まで幅広いライフスタイル商品を展開しています。

- **URL**: https://bruno-onlineshop.com/
- **取扱ブランド**: BRUNO（メイン）、MILESTO（トラベル用品）
- **価格帯**: 約2,600円〜30,000円（ギフト含む）
- **会員プログラム**: BRUNO POINT SERVICE（BPS） ／ ポイント還元率3%〜

## 2. サービス・ブランドの特徴

### ブランドコンセプト
「暮らしを整え直す」をテーマに、毎日の生活に寄り添うキッチン家電・インテリア雑貨・美容アイテムを提供。デザイン性と機能性を両立した商品ラインナップが特徴。

### 主な強み
| 特徴 | 詳細 |
|------|------|
| デザイン性の高さ | 北欧調のシンプルでおしゃれなデザイン。インテリアとしても映える家電 |
| 豊富なカラーバリエーション | パステルカラーなど幅広い色展開でギフトにも最適 |
| 1人暮らし〜ファミリー対応 | コンパクトサイズから大型まで、ライフステージに合わせた商品展開 |
| ギフト需要への対応 | カタログギフト・体験ギフトなどギフト商品が充実。結婚・出産・新築祝い等 |
| 会員ランク特典 | 半年ごとのランク更新制度。プラチナ会員は送料無料・高還元率 |
| 定期キャンペーン | 「新生活2BUY10%OFF」「会員ランク更新キャンペーン」など季節連動施策 |

### 商品カテゴリ
- **キッチン**: ホットプレート、ホットサンドメーカー、トースター、ブレンダー、圧力クッカー、電気ケトル、コーヒーメーカー、食器

- **ピクニック**: 弁当箱、水筒
