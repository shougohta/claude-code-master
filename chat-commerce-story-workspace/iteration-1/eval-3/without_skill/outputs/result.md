## 4. 購買につながるチャット導線設計

### 設計方針
- ユーザーの**購入目的（自分用 or ギフト用）**を最初に分岐させる
- ギフト用は**用途・予算・相手**で絞り込み、最短3ステップで商品提案
- 自分用は**ライフスタイル・シーン**から欲求を引き出し購買意欲を高める
- **人気商品・限定キャンペーン**を自然に組み込み購入を後押しする
- 決済直前で**会員登録メリット・まとめ買い割引**を訴求する

## 7. 各フローの設計ポイント

| ポイント | 内容 |
|----------|------|
| 最初の分岐 | 自分用 vs ギフト用で異なるUXを提供し、離脱を防ぐ |
| ギフトフロー | 用途→予算→商品提案の3ステップで最短購買を実現 |
| 自分用フロー | シーン・ライフスタイルから訴求し、欲求を具体化 |
| クロスセル | 2BUY10%OFFキャンペーンをカート追加後に自然に提示 |
| 会員化 | 購入直前に会員登録メリットを訴求しLTV向上 |
| ギフト付加価値 | ラッピング・熨斗・直送オプションを提案し購買障壁を下げる |
| 緊急性 | 期間限定キャンペーン・セール価格を明示し購買を促進 |

## 8. KPI設計（推奨）

| 指標 | 目標値 | 測定方法 |
|------|--------|----------|
| チャット開始率 | サイト訪問者の15%以上 | チャット起動数 / PV |
| チャット完了率（購入フロー到達） | 40%以上 | 決済ページ遷移数 / チャット開始数 |
| チャット経由購入率（CVR） | 5%以上 | チャット経由注文数 / チャット開始数 |
| ギフト用ラッピング追加率 | 30%以上 | ラッピング追加数 / ギフトフロー完了数 |
| 会員登録率（チャット経由） | 25%以上 | チャット経由会員登録数 / チャット購入完了数 |

---

## 9. チャット導入における推奨事項

1. **エントリーポイント**: サイトトップ画面の右下に常時表示するチャットウィジェット
2. **起動タイミング**: 初回訪問30秒後または商品ページ滞在15秒後に自動ポップアップ
3. **スマホ対応**: モバイル比率が高いため、スマートフォンでの操作性を最優先に設計
4. **AIチャット連携**: 定型フロー外の質問はFAQデータベースやAIで対応し、離脱を防止
5. **A/Bテスト**: ウェルカムメッセージ・ボタン文言・提案順序を定期的に検証・改善
6. **季節対応**: 新生活（3〜4月）・夏ギフト（7〜8月）・冬ギフト（11〜12月）は専用フローを用意
