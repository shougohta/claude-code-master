# チャットコマース ストーリーパターン設計 Skill

ECサイトのURLを分析し、購入まで誘導するチャットストーリーの設計書を生成する。

## ステップ1: サイト調査

URLからサイト情報を取得し、以下を把握する：

1. **WebFetch** でトップページを取得
2. さらに必要に応じて以下のページも取得：
   - `/products`、`/items`、`/ranking`、`/bestsellers` などの商品一覧・ランキングページ
   - `About` や `会社概要` ページ（サービス概要把握のため）
3. 収集する情報：
   - **ジャンル**: 何を売っているEC（アパレル、化粧品、食品、家電など）
   - **概要**: ブランドのコンセプト・ターゲット層・価格帯
   - **特徴**: 他サービスとの差別化ポイント（送料無料、会員特典、限定商品など）
   - **人気商品ランキング**: サイト内の売れ筋・人気・おすすめ商品（上位5〜10品）

> サイトによっては売上ランキングが非公開の場合がある。その際は「注目商品」「おすすめ」「新着」などから代替して推定ランキングとして記載する。

## ステップ2: ストーリーパターン設計

調査結果をもとに、以下の3パターンのチャットストーリーを設計する。

各パターンは「ユーザーの状態と目的」に合わせた会話フローで構成する。会話は**ボットの発言**と**ユーザーの選択肢（ボタン）**の形式で記述する。

### パターン共通の設計原則

- 最初の一言でユーザーの興味を引く（押しつけがましくならないように）
- 選択肢は2〜3個に絞り、ユーザーが迷わないようにする
- 各ステップは短く、テンポよく進む
- 最終的に「商品ページへの誘導」または「カートへの追加」につながるようにする
- そのサービスのジャンル・トーン・ブランド感に合わせた言葉遣いにする

### パターン①: 初回訪問者向け

**目的**: サービスを知らないユーザーにブランドを紹介し、最初の購入に繋げる

設計のポイント：
- ブランドの世界観・強みを簡潔に伝える
- 「何を買えばいいかわからない」という不安を解消する
- 人気商品・入門商品を提示して購入ハードルを下げる
- 初回限定クーポンや特典があれば訴求する

### パターン②: リピーター向け

**目的**: 再購入・クロスセル・アップセルを促す

設計のポイント：
- 「またきてくれた」感謝・親しみのある出迎え
- 前回の購入カテゴリに関連する新商品・補充商品を提案
- ランキング上位商品や季節限定品を紹介
- 次回購入につながる特典（ポイント、会員特典）を訴求

### パターン③: VIP向け

**目的**: 高額・高頻度購入ユーザーのロイヤリティを高め、継続購入・限定品購入を促す

設計のポイント：
- VIPであることへの特別感を演出する
- 限定商品・先行販売・プレミアムラインを優先提示
- 専任スタッフ対応や特別サービスがあれば強調
- 感謝と信頼関係を前面に出したトーンにする

## ステップ3: 出力形式

以下の構成でMarkdownドキュメントを生成する：

```
# [サービス名] チャットストーリー設計書

## 1. サービス調査結果
### ジャンル
### サービス概要
### 特徴・差別化ポイント
### 人気商品ランキング（上位5〜10品）

## 2. チャットストーリーパターン

### パターン①: 初回訪問者向け
[ボットとユーザーの会話フロー]

### パターン②: リピーター向け
[ボットとユーザーの会話フロー]

### パターン③: VIP向け
[ボットとユーザーの会話フロー]
```

### 会話フローの記述例

```
🤖 ボット: はじめまして！〇〇へようこそ😊
　　　　　 まず、どんなアイテムをお探しですか？

👤 選択肢:
　 [人気ランキングを見る]
　 [カテゴリから探す]
　 [おすすめを教えて]

---（[人気ランキングを見る] を選択した場合）---

🤖 ボット: 今一番人気の商品はこちらです✨
　　　　　 「〇〇〇〇」¥3,980
　　　　　 ▶ [商品を見る]

👤 選択肢:
　 [カートに追加する]
　 [他の商品も見る]
　 [もう少し詳しく教えて]
```

## 注意事項

- WebFetchで取得できなかった情報は「情報取得不可」と明記し、スキップして進める
- サイトのブランドトーン（カジュアル、高級感、親しみやすさ）に合わせてチャットの文体を調整する
- 絵文字の使用はブランドトーンに合わせて判断する（高級ブランドは控えめに）
- ユーザーの実際の購入体験に沿った自然な会話になるよう心がける
