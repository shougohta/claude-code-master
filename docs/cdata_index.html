CData とは

CData Software は、250以上のデータソースに対して標準化されたドライバー・コネクターを提供するデータコネクティビティプラットフォームです。アプリケーションや BI / ETL ツールから API・クラウドサービス・データベースを SQL ライクに透過的に扱えるようにすることを主眼としています。

Selection Guide

選択の指針

ユースケース別に最適なサービスが異なります。

CData が向く場合
既存ツールから直接接続したい
Tableau・Power BI などの BI ツールや、既存の ETL から SaaS API を SQL で透過的に参照したい場合。ドライバーレベルの柔軟性が必要な場合。
Fivetran / Airbyte が向く場合
DWH へ安定的にデータを同期したい
BigQuery・Snowflake などへデータを定期同期したい場合。パイプラインの安定稼働・監視を重視する場合。Airbyte はコスト重視のチームにも適する。
MuleSoft / Informatica が向く場合
大規模統合基盤を構築したい
基幹システム連携、API 管理、データガバナンスまで含めた大規模なエンタープライズ統合基盤が必要な場合。予算規模が大きい組織向け。
